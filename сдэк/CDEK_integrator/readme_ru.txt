	===================================================
	Формат обмена данными «СДЭК» - Интернет-Магазин
		Спецификация требований
	===================================================

Описание
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Файл архива содержит документы и описание необходимых требований 
для обмена данными между Компанией СДЭК и Интернет-магазином.
По всем вопросам интеграции стоит обращаться на адрес integrator@cdek.ru. Если доставка между разными странами, первоначально стоит проконсультироваться с Вашим менеджером, чтобы однозначно определить каким форматом обмена данными пользоваться.	
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Входящие файлы
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
- СДЭК_Формат обмена данными_ru.doc: описание регламента обмена данными внутри страны (Доставка внутри России, Казахстана) на русском, примеры;
- CDEK_Protocol of integration_en.doc: описание регламента обмена данными внутри страны (Доставка внутри России, Казахстана) на английском, примеры;
- СДЭК_Международный формат обмена данными_ru.doc: описание регламента обмена данными среди разных стран (например, ввоз груза из США в Росиию, с прохождением таможни) на русском, примеры;
- CDEK_Protocol of foreign integration_en.doc: описание регламента обмена данными среди разных стран на английском (например, ввоз груза из США в Росиию, с прохождением таможни) на английском, примеры;
- City_XXX_YYYYMMDD.xls, где XXX – трехбуквенный код страны, YYYYMMDD – дата формирования файла: список городов определенной страны базы СДЭК, содержит идентификаторы, название, ограничение по сумме наложенного платежа;
- OrderPrint.pdf: пример печатной формы квитанции к заказу;
- Invoice.pdf: пример печатной формы квитанции к международному заказу;
- ErrorCode.xls: коды и описание ошибок;
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
