	===================================================
	Data interchange format "CDEK" - online Store
		Specification requirements
	===================================================

Description
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
The archive file contains the documents and description of the necessary requirements to exchange data between the Company CDEK and online store.
On all questions of integration should apply to the address integrator@cdek.ru. If shipping between different countries, initially you should consult with Your Manager to clearly define what data format to use.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Incoming files
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
- СДЭК_Формат обмена данными_СДЭК_ru.doc: description of the rules communication within the country (delivery within Russia, Kazakhstan) in russian, examples;
- CDEK_Protocol of integration_en.doc: description of the rules communication within the country (delivery within Russia, Kazakhstan) in english, examples;
- СДЭК_Международный формат обмена данными_ru.doc: description of schedule of data exchange among different countries (for example, importing goods from the USA to Russia with customs clearance) in russian, examples;
- CDEK_Protocol of foreign integration_en.doc: description of schedule of data exchange among different countries in English (for example, importing goods from the USA to Russia with customs clearance) in english, examples;
- City_XXX_YYYYMMDD.xls, where XXX is the three-letter country code, YYYYMMDD is the date of formation of the file: list of cities in a specific country base SDEK, contains IDs, name, limit the amount of cod;
- OrderPrint.pdf: example of a printed form of receipt to the order;
- Invoice.pdf: example of a printed form of receipt to the international order;
- ErrorCode.xls: codes description of the fault;
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
